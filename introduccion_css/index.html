¿Qué es CSS y para que sirve?

CSS (Cascading Style Sheets) es un lenguaje de programación que se utiliza para definir cómo se ve una página web. También se conoce como Hojas de Estilo en Cascada.


¿Qué hace CSS?

  • Define la presentación visual de un documento HTML
  • Permite cambiar el aspecto de cualquier elemento de una página web
  • Separa la estructura del documento HTML de su presentación
  • Permite realizar cambios en la apariencia de una web sin afectar al contenido
  • Permite presentar el mismo documento en diferentes estilos
